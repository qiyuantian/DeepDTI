# DeepDTI

**1. s_GDSI_tutorial.m**

Step-by-step GDSI reconstruction of diffusion PDF and ODF from multishell voxel data in MATLAB.

**2. shelldens.m**

  MATLAB function to compute sampling density nonuniformity correction factor for multishell geometry.
